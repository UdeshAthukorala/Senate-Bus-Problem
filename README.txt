How to run the Programme
======================================================

Method 1:

1. Open 'SenateBusProblem' project folder in an IDE which supports Java (i.e IntelliJ)
2. Then Run Main.java file, which includes main class.

-------------------------------------------------------

Method 2: 

1. Open a command prompt window and go to the directory which includes source files(.\SenateBusProblem\src)
2. Then type & execute following commands

	javac Bus.java
	javac BusScheduler.java
	javac Resources.java
	javac Rider.java
	javac RiderScheduler.java
	javac Main.java
	Main

--------------------------------------------------------
[Source Files are including in the following folder(.\SenateBusProblem\src)]
